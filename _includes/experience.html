Student Research Assistant

July 2015 - January 2017
Hochschule für Technik und Wirtschaft Dresden

The objective was to automatically classify regions within occupancy grid maps (e.g., identifying 'floor' vs. 'obstacle') based on local geometric patterns. My core responsibilities involved developing a complete data processing and modeling pipeline in Python. This included researching suitable deep learning (Theano, Lasagne, Nolearn) and data handling/visualization libraries (Scipy, Matplotlib), and implementing a script to extract overlapping local patches from large grid maps. I engineered features for these patches by calculating the Euclidean Distance Transform (EDT) on binarized versions, capturing proximity information, and automated the labeling process to create datasets for training. Subsequently, I implemented, trained, and systematically evaluated various neural network architectures, including Multi-Layer Perceptrons (MLPs) and Convolutional Neural Networks (CNNs), using Lasagne and Nolearn. This involved experimenting with network structures, filter sizes, applying regularization techniques like Dropout, and utilizing data augmentation (random flipping). Finally, I developed scripts for performance analysis (accuracy, confusion matrices) and visualization of learned features, network activations, training progress, and the final reconstructed semantic map, contributing to the overall evaluation and understanding of the deep learning approach for this task.
